# Civic-sense Web Application
---
Civic-sense Web Application is a platform that allows citizens to report civic issues such as potholes, streetlight failures, sanitation problems, and other public infrastructure concerns. The application provides a user-friendly interface for reporting issues, tracking their status, and receiving updates from local authorities.
The web application is built using ReactJS and Vite for a fast and efficient development experience. It includes features such as user authentication, issue reporting, status tracking, and an admin dashboard for local authorities to manage and resolve reported issues.
